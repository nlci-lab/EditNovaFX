# Release Notes

## v0.0.1-alpha (2026-03-12)

Public pre-release alpha build of EditNovaFX.

### Download
- Installer: EditNovaFX_Setup_v0.0.1-alpha.exe

### Includes
- Multi-track timeline editing (video, audio, subtitles)
- Subtitle import and editing (SRT)
- Video export using FFmpeg
- AI transcription support (local and cloud)
- YouTube publishing workflow

### Notes
- This is an early test build and may contain bugs.
- Recommended for testers and early adopters.
- Some features need extra setup (FFmpeg, Whisper, YouTube API).
